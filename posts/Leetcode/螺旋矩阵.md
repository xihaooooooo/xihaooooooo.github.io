[54. 螺旋矩阵 - 力扣（LeetCode）](https://leetcode.cn/problems/spiral-matrix/description/)
## 1. 三段式记忆法（核心流程）

你可以把整个算法拆解为三个独立的模块，每个模块都是链表的基础操作：

### 第一步：找中点 (Find Middle)

使用 **快慢指针**。快指针走两步，慢指针走一步。

- **目的**：把链表切成前后两半。
    
- **口诀**：快二慢一，慢指针最后就是中点。
    

### 第二步：翻转后半部分 (Reverse Second Half)

将从中点开始的后半部分链表进行原地翻转。

- **目的**：为了能从末尾向前取节点（链表是单向的，只能通过翻转来实现反向访问）。
    
- **口诀**：三指针翻转（`pre`, `cur`, `nxt`）。
    

### 第三步：交替合并 (Merge / Zip)

像“拉拉链”一样，左边取一个，右边取一个，连接起来。

- **目的**：实现 $L_0 \rightarrow L_n \rightarrow L_1 \rightarrow L_{n-1} \dots$
    
- **口诀**：存下 Next，交叉连接。
    

---

## 2. 图解理解

假设链表为：`1 -> 2 -> 3 -> 4 -> 5 -> 6`

1. **找中点**：找到 `4`（如果是奇数就是正中间，偶数是后半段开头）。
    
2. **翻转后半段**：`4 -> 5 -> 6` 变成 `6 -> 5 -> 4`。
    
3. **交替合并**：
    
    - 左半段：`1 -> 2 -> 3`
        
    - 右半段：`6 -> 5 -> 4`
        
    - 合并结果：`1 -> 6 -> 2 -> 5 -> 3 -> 4`

## 3. 合并逻辑的“背诵模板”

第三步的 `while` 循环是很多人的难点，建议用“存-连-移”三字诀：

Java

```
while (head2.next != null) {
    // 1. 存：先把两边的后继节点存起来，防止断链
    ListNode nxt = head.next;
    ListNode nxt2 = head2.next;

    // 2. 连：交叉连线
    head.next = head2;    // 左连右
    head2.next = nxt;     // 右连左的下一个

    // 3. 移：两个指针同时后移，准备下一对
    head = nxt;
    head2 = nxt2;
}
```

---

## 4. 为什么这个代码能“快速背诵”？

因为它是**组合题**。

- `middleNode` 是 LeetCode 876 原题。
    
- `reverseList` 是 LeetCode 206 原题。
    
- 如果你已经掌握了这两个基础，你只需要背那几行 **“拉链合并”** 的逻辑即可。
    

### 避坑小贴士：

- **为什么循环条件是 `head2.next != null`？**
    
    因为在翻转后，原中点的 `next` 指向了 `null`。当 `head2.next` 为空时，说明所有节点都已经处理完毕。
    
- **空间复杂度**：这种解法是 $O(1)$ 的，因为它是原地修改，不需要额外的数组存储。
